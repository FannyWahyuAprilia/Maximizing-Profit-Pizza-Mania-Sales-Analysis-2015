# Maximizing-Profit-Pizza-Mania-Sales-Analysis-2015
Full BI analysis of Pizza Mania's 48.6K orders across 4 dimensions: temporal patterns, product performance, revenue contribution, and strategic recommendations. Identified peak hours (12–13 &amp; 18–19), top product (Classic Deluxe), and highest revenue generator (Thai Chicken)

## Project Overview

This project analyzes pizza sales data using SQL in Google BigQuery and visualizes the results through Looker Studio dashboards. The analysis aims to identify sales performance, customer purchasing patterns, and top-performing pizza categories.

## Objectives

* Analyze overall sales performance
* Identify best-selling pizza categories
* Measure revenue contribution by pizza type
* Understand customer purchasing trends
* Create an interactive dashboard for business decision-making

## Tools Used

* Google BigQuery
* SQL
* Looker Studio
* GitHub
